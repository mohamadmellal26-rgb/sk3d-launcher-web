SK3D Launcher Frontend

This project is the frontend interface for the SK3D Launcher system. The project aims to provide a simple and easy user interface for users to download and manage the application.

Features:

Modern and responsive user interface.

System for displaying download links based on the platform.

Easy integration with any backend system.

Technologies Used:

React

TypeScript

CSS Modules

How to run:

Ensure Node.js is installed on your machine.

Install the required dependencies:
npm install

Start the development environment:
npm run dev

Project Structure:

/src/components: Basic UI components.

/src/pages: Main pages.

/src/styles: Styling files.
